# Harutyun Grigoryan Portfolio

Static portfolio website for R&D engineering, AI automation, backend systems, and production video demonstrations.

## Structure

```text
.
├── index.html
├── assets/
│   ├── video-posters/
│   └── videos/
└── README.md
```

## Deployment

This project can be hosted directly with GitHub Pages because it is a static HTML site.

Use `index.html` as the entry point and keep media assets under `assets/`.
